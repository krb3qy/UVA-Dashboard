User Guide✕

Queue Tabs

  • Each tab watches one or more queues. Click + Queue to open Settings and add queues.
  • Right-click a tab to Edit queues, Rename, or Remove it.
  • Groups (2+ queues in one tab) show a blue bar on the left. Hover for the full member list.

My Interactions / Call History

  • My Interactions: live calls you're on. Select a row, then use the bottom toolbar for Pickup / Hold / Mute / Transfer / etc.
  • Call History: last 14 days of your conversations.

Queue Panel

  • Shows every live call in the active tab's queue(s).
  • Right-click any column header to show/hide columns for that queue.
  • Drag a row onto another row to merge the two calls into a conference.

Workgroup Statistics

  • Aggregated across every queue in the active tab (users summed, longest-wait maxed).

Directory

  • Members of the active tab's queues, sortable by name / presence / routing status.
  • Each agent's dot uses GC's routing status when they're actively on queue. When their routing status is OFF_QUEUE, the dot falls through to their presence instead — so an Available agent reads green, not generic grey "Off Queue."

Routing-status states (agent is on queue):

Dot Label GC routing status Meaning
On Queue IDLE On queue and ready to take a call.
Interacting INTERACTING Handling an ACD interaction (queue-routed call) — including ACW wrap-up.
Communicating COMMUNICATING Handling a non-ACD interaction (direct dial, internal extension, transfer-in).
Not Responding NOT_RESPONDING An ACD call was offered and the agent didn't answer in time. Routing skips them until cleared.

Off-queue states (routing status is OFF_QUEUE; dot reflects presence):

Dot Label GC presence Meaning
Available AVAILABLE Signed in but not on queue. Could go on queue.
Busy BUSY / MEETING Self-selected busy.
Away / Break / Meal / Training AWAY / BREAK / MEAL / TRAINING Stepped away. Won't receive calls.
Offline / Off Queue OFFLINE / OFF_QUEUE Not signed in or explicitly off queue.

Conference Dialer

  • Toolbar button or Actions > Conference Dialer. Add numbers, save a group, click Bridge All.
  • Outbound CID defaults to your personal line; pick a queue to present that queue's number instead.

Queue Alerts

  • Configure under File > Alert Settings. Add a rule per queue with its own sound and banner color.
  • When a call enters a watched queue, a flashing banner in the rule's color appears with the rule's sound. Click the banner's Pick Up to answer, or Dismiss to silence.
  • The status-bar Alerts chunk shows on/off based on the master toggle and whether any rule with a queue is enabled. Hover for the active rules at-a-glance.

Workspaces

  • Save your current layout (tabs, columns, splitter sizes, sidebar state) via Workspaces > Manage Workspaces. Switch layouts from the Workspaces menu.

Popout

  • Toolbar Pop Out opens the dashboard in its own window. Token carries over via localStorage.
